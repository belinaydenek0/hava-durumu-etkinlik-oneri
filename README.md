# Akıllı Hava Durumu ve Etkinlik Asistanı

Bu proje, mevcut hava durumuna göre kullanıcı tercihlerini analiz ederek kişiselleştirilmiş etkinlik önerileri sunan bir Python & MySQL uygulamasıdır.

## Kurulum
1. `pip install requests mysql-connector-python python-dotenv` komutunu çalıştırın.
2. `.env` dosyası oluşturup `WEATHER_API_KEY` ve `DB_PASSWORD` bilgilerinizi girin.
3. SQL dosyasını veritabanınıza import edin.

## Özellikler
- Kullanıcı tercihlerine göre filtreleme.
- OpenWeatherMap API entegrasyonu.
